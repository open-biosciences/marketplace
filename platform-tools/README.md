# Platform Tools

Developer-facing platform engineering tools for the Open Biosciences ecosystem.

## Commands

| Command | Description |
|---------|-------------|
| `scaffold-fastmcp` | Generate a new FastMCP server project with standard structure |
| `scaffold-fastmcp-v2` | Generate a FastMCP v2 server with enhanced patterns |

## Skills

| Skill | Description |
